# انقلاب — Inquilab

نسخة ويب عربية محلية من لعبة الخداع والاستنتاج الاجتماعي المستوحاة من Coup، بواجهة خرافية داكنة ونظام أدوار Hot-seat من 2 إلى 6 لاعبين.

## المزايا

- واجهة عربية RTL بطابع خرافي وتشويقي.
- إعداد 2–6 لاعبين مع أسماء قابلة للتعديل.
- توزيع تلقائي للبطاقات والعملات.
- بطاقات مخفية ومكشوفة، وخروج اللاعب عند فقدان التأثيرين.
- إجراءات الدخل، المساعدات الأجنبية، الضرائب، الاغتيال، السرقة، والتبادل.
- نظام تحدي الادعاءات.
- نظام منع المساعدات/السرقة/الاغتيال مع إمكانية تحدي المنع.
- سجل أحداث فوري لتتبع ما حدث في المجلس.
- نافذة خاصة لعرض بطاقات اللاعب الحالي قبل تمرير الجهاز.
- صور كروت مدمجة لكل الشخصيات: القبطان، السفاح، الدوق، الكونتيسة، والسفير.

## التشغيل المحلي

افتح الملف `index.html` مباشرة في المتصفح، أو شغّل خادماً محلياً:

```bash
python3 -m http.server 5173
```

ثم افتح:

```text
http://localhost:5173
```

## ملاحظات القواعد

تم اعتماد منطق المنع وفق الجدول التفصيلي في المواصفات: الدوق يمنع المساعدات الأجنبية ولا يأخذ العملات لنفسه، والقبطان/السفير يمنعان السرقة، والكونتيسة تمنع الاغتيال.
